간단한 채굴 방법

  1. 회원 가입을 하세요
  2. 자동 환전 페이지에서 최종적으로 받고 싶은 코인을 선택하세요. (선택사항)
  3. 채굴을 시작하세요
    • 단순 코인 채굴 (포트 20XXX)
      왼쪽 아래 표에서 원하는 코인 옆에 쓰여있는 포트 번호를 확인하세요. 알맞은 마이너를 이용해서 해당 포트로 바로 채굴하시면 됩니다.
    • 알고리즘 스위치 채굴 (포트 17XXX)
      마이닝 풀 허브에서는 각 알고리즘별로 스위치할 수 있는 포트를 제공합니다. 이 포트들은 자동으로 그 때 그 때 가장 수익률 높은 코인을 채굴합니다. 해시를 렌탈하거나 ASIC, FPGA를 이용하는 마이너에게 추천합니다.
    • 멀티 알고리즘 스위치 채굴 (포트 12XXX)
      허브 기능을 이용해 여러 알고리즘의 코인들도 스위치하면서 채굴할 수 있습니다. GPU 채굴자들에게 좋지만 설정 방법이나 최적화 방법이 비교적 어렵습니다.
  4. 채굴 최적화에 더 알아보세요.
    초기 difficulty를 지정하는 법 (영어)
  • 풀 수수료는 0.9% 입니다. 다른 어떤 풀보다도 낮습니다.
