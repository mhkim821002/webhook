참고

이 페이지는 모든 코인의 잔고를 표시합니다.

자동 환전용 코인을 지정하면, 새로 채굴되는 코인들은 모두 환전용으로 적립됩니다.

각 코인의 "지갑 관리" 링크에서 "일반 지갑"과 "자동 환전용 지갑" 자금을 상호 이동시킬 수 있습니다.

A -> B 로 환전하는 경우 환전용 지갑(A) -> 환전 중(A) -> 일반 지갑(B) 순서로 자금이 이동하게 됩니다.

환전하기에 너무 작은 금액이어서 환전이 불가능한 상태로 오랫동안 남아있는 경우에는 원하는 코인으로 환전받지 못할 수도 있습니다. 이런 경우 기존 코인으로 돌려받거나 비트코인으로 적립시켜드립니다.

현재 페이지에 나오는 잔고 숫자들은 일정 시간 간격으로 갱신됩니다. 각 풀에서 보여주는 잔고는 실시간 값이므로 서로 불일치할 수 있습니다.
